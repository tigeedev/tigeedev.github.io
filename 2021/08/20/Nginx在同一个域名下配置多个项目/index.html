Nginx在同一个域名下配置多个项目

充电学习 nginx 服务器 2021/08/20

前言

服务器可以部署多个项目，在只有一个域名的情况下，可以给不同的项目分配不同的二级域名来进行访问。

配置过程

1. 添加二级域名

  • 添加A记录到主机
  • 同一个域名可以解析出N个二级域名

2. 配置nginx

  • nginx 中另添加一个 server
  • 修改后重启nginx
    • 先使用 nginx -t 测试语法是否正确
    • 使用命令 nginx -s reload 重新加载配置文件
